**"IV 压缩陷阱"机制：**

```

          ↓
```

关键结论：**损失不是因为 SPX 持续下跌，而是因为 bear market 中期反弹压缩 IV，导致 Δ0.20 选出的 strike 距离 spot 不足，再次暴跌时被穿透。**

## 七、结构性发现

### "IV 压缩陷阱"是主要尾部风险

熊市中的典型路径：
```
大跌 → 反弹 → IV 压缩 → Δ0.20 选出过近 strike → 再次大跌 → 击穿
```

这不是"随机"的尾部事件，而是有结构性的：**只在 bear market 内的技术性反弹后，IV 短暂被压缩时才会出现。**

纯牛市 + 低波动期（2023–2024）：Δ0.20 也只有 2–3% OTM，但 realized move 同样很小，strategy 完美运行。

### 风险管理含义

| 方案 | 有效 | 原因 |
|------|------|------|
| ❌ OTM% 或 IV 门槛（static filter） | 否 | 会剔除大量盈利 cycle |
| ✅ **仓位规模控制**（position sizing） | 是 | 单 cycle 最大亏损 −2.84%；只要头寸不过大，全年可覆盖 |
| ✅ **动态 delta 或 minimum OTM 检查**（可选） | 有限 | 如规定"K 距 S 不得低于 4%"，避免 Aug-22 型 cycle；但需验证不剔除太多好 cycle |
| ✅ **月度 cycle 不加仓**（不叠加头寸） | 是 | 防止多个 cycle 同时亏损放大 |
