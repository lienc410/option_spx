## 二、Phase A — 隐含移动溢价（核心统计检验）

**方法：**
- 入场日 T 前，找最近覆盖财报的期权到期日（earnings expiry）
- T-1（或 T-3）当日：取 ATM straddle = ATM call + ATM put（同一 expiry）
- Implied Move = straddle_price / S_entry
- Realized Move = |S(T+1) - S_entry| / S_entry
- Premium = Implied Move - Realized Move（正值 = 期权市场高估财报波动）

## 八、已知局限性

| 局限 | 说明 |
|------|------|
| 财报日期硬编码 | 使用已知历史日期，与 yfinance 获取失败无关。少数日期可能偏差 1 天（如 AMC 财报到 DMO 影响）|
| DTE 极短（1–4 天）| 所有入场均使用最近周度到期，T+1 结算几乎无时间价值缓冲，最差情况 = 完全内涵价值结算 |
| 无 bid/ask | 用 close 近似，实际成交价更差；铁鹰 4 腿结构滑点可达 $2–4 per event |
| META 排除须稳定规则 | 不能在实盘中因为"META 过去跳动大"就主观跳过；需要明确的 IV crush 统计评分作为入场过滤 |
| N 较小 | 每个标的仅 8–15 事件；COST/JPM 的 Sharpe 优异仍在置信区间内 |
| 年度不稳定性 | 2024 年全体溢价接近 0；策略在低波动牛市中可能无效 |
| 4 年窗口 | 含 2022 熊市 + 2023–2025 牛市，方向性偏差影响 IV crush 统计 |
