# Q066 — Aftermath vs Q042 Co-firing Frequency

**Owner**: Quant Researcher
**Status**: 完成；**结论：两个 addon empirically low-overlap and structurally non-redundant，不应合并**
**2nd Quant Review**: [task/q066_cofiring_2nd_quant_review_packet_2026-05-12_Review.md](../../task/q066_cofiring_2nd_quant_review_packet_2026-05-12_Review.md) — PASS WITH CAVEAT

## Question

PM 询问 SPEC-064 Aftermath broken-wing IC 与 Q042 Drawdown Overlay 是否功能重复（"都是抄底入场"）。已有 [doc/addon_greek_orthogonality_2026-05-12.md](../../doc/addon_greek_orthogonality_2026-05-12.md) 论证两者 Greek 正交；本研究 empirically 量化 2007-2026 触发日 / 事件级 co-firing 频率，确认正交性是真实数据特征而非纯理论。

## TL;DR

| 维度 | 结果 | 结论 |
|---|---|---|
| **日级 same-day 重叠** | 5 / 553 days = **0.9%** | 几乎零同步 |
| **Q042-A 事件 ±5 TD co-fire with Aftermath** | 9 / 35 = **26%** | 74% Q042-A 触发是 vol-quiet drawdown，aftermath 不会触发 |
| **Q042-B 事件 ±5 TD co-fire with Aftermath** | 4 / 5 = **80%** | 但 N 仅 5，B 是深度崩盘信号天然伴随 VIX spike |
| **Aftermath windows ±5 TD co-fire with Q042-A** | 13 / 90 = **14%** | 86% aftermath windows 无 SPX 深度回撤伴随 |
| **Aftermath windows ±5 TD co-fire with Q042-B** | 6 / 90 = **7%** | 93% aftermath windows 无 B 级深崩 |

**结论**：日级近乎零重叠，事件级 74-86% 异步。两个 addon 在 19 年实证中**捕捉了显著不同的机会集**。**The two addons are empirically low-overlap and structurally non-redundant — 不应合并 / 不应竞争**。

> **Wording caveat（per 2nd Quant Review）**：不使用 "fully orthogonal" 表述。Greek sign opposition supports non-redundancy 但 does not prove portfolio hedge effectiveness（详 §6.4 + §6.6 co-loss failure mode）。

## Method

[q066_cofiring.py](q066_cofiring.py) 完整复现两个 addon 的生产触发逻辑：

**Aftermath** ([strategy/selector.py](../../strategy/selector.py:295) `is_aftermath()`):
- trailing 10-TD VIX peak ≥ 28
- current VIX ≤ peak × 0.90
- current VIX < 40

**Q042 Sleeve A** ([signals/q042_trigger.py](../../signals/q042_trigger.py)):
- `ddATH ≤ -4%` 且 `armed=True` → 触发；mock 30-TD hold（DTE）
- re-arm: `ddATH ≥ -2%` after position closes

**Q042 Sleeve B**:
- Outer: `ddATH ≤ -15%` → enter watching
- Inner: 30-TD watch 内首次 close > MA10 → 触发；mock 90-TD hold
- re-arm: `ddATH ≥ -2%`

数据：yfinance ^VIX + ^GSPC 2007-01-03 → 2026-05-12 (4870 trading days)

## Findings

### 1. 触发日数量（19 年）

| Addon | 触发日数 | 年化频率 |
|---|---|---|
| Aftermath | 518 | ~27 / 年 |
| Q042-A trigger | 35 | ~1.8 / 年 |
| Q042-B watching | 32 | ~1.7 / 年 |
| Q042-B trigger | **5** | ~0.26 / 年（一代人 2-3 次）|

注意：Aftermath 是 daily evaluation 持续在 26-day 窗口期；Q042 是离散事件。所以频率口径不同。但**两者触发日的 same-day 重叠是直接可比的**。

### 2. 日级 confusion matrix

|  | Q042-A or B trigger = YES | Q042 = NO | 行总 |
|---|---|---|---|
| Aftermath = YES | **5** | 513 | 518 |
| Aftermath = NO | 35 | 4317 | 4352 |

- Same-day overlap rate（任一触发的日子）= 5 / (5+513+35) = **0.9%**
- 即使 conditional on either firing 那天，**99% 的日子只有一个 addon 触发**

### 3. 事件级 co-firing（±5 TD window）

**Q042-A 视角**（35 events）：
| 类别 | count | % |
|---|---|---|
| co-fire with Aftermath ±5 TD | 9 | 26% |
| **vol-quiet drawdown（aftermath 不触发）** | **26** | **74%** |

**74% Q042-A 触发是 aftermath 永远不会捕捉到的机会**。代表事件：2013-06-20 / 2013-08-27 / 2014-01-29 / 2014-10-09 / 2014-12-15 / 2015-01-30 / 2015-11-13 / 2015-12-31 / 2016-05-19 / 2018-10-10 / 2019-05-13 / 2019-08-05 / 2019-10-02 / 2024-04-17 / 2024-07-24 / 2025-01-10 / 2025-02-27 / 2025-11-20 等。这些日子 VIX 偏低（16-26），aftermath gate 不触发，但 SPX -4% drawdown 是 real signal。

**Q042-B 视角**（5 events）：
| 日期 | ddath_% | VIX | 距最近 aftermath |
|---|---|---|---|
| 2008-01-28 | -13.49% | 27.78 | 0 TD（同日）|
| 2018-12-31 | -14.46% | 25.42 | 0 TD |
| 2020-03-25 | -26.89% | 63.95 | 13 TD（aftermath 被 VIX ≥ 40 拦截）|
| 2022-05-17 | -14.75% | 26.10 | 0 TD |
| 2025-04-09 | -11.19% | 33.62 | 0 TD |

B trigger N=5 偏小但 **80% co-fire** —— 因为 ddATH ≤ -15% 的深度崩盘几乎一定伴随 VIX spike & off-peak。但其中 2020-03-25 这例特殊：Q042-B 触发但 aftermath 被 VIX ≥ 40 EXTREME 拦截（即 [Q065](../q065/q065_memo_2026-05-12.md) 所说的"保护机制正确工作"）。

> **2nd Quant Review Q7.2 caveat**：**Q042-B co-fire rate is not statistically stable due to N=5 and should be monitored as more events accumulate**。Treat B as "monitor as sample grows," not as evidence of redundancy。B 的逻辑（深 drawdown + MA10 reclaim）与 aftermath VIX structure 仍非同一触发器，但 80% co-fire 比例本身不构成强结论。

**Aftermath 视角**（90 windows）：
- 13 / 90 (14%) co-fire with Q042-A trigger ±5 TD
- 6 / 90 (7%) co-fire with Q042-B trigger ±5 TD
- **86-93% aftermath windows 期间 SPX 没有 -4% 回撤** —— aftermath 在很多 "VIX 回落但 SPX 没大跌" 的场景下独立工作

### 4. 同时触发的事件集

5 个 day-level overlap + 同期 ±5 TD 的所有事件：
- 2007-12-04（A）
- 2008-01-28（B）
- 2015-08-20（A，VIX 19, +3 TD from aftermath）
- 2018-02-05 Volmageddon（A, VIX 37）
- 2018-12-31（B）
- 2020-02-24（A）
- 2020-09-04 / 2020-10-21（A，post-COVID rolling ATH 反复 -4%）
- 2021-03-04 / 2021-12-01（A）
- 2022-05-17（B）
- 2025-04-09（B，tariff）
- 2026-03-12（A）

这些事件的共性：**VIX 上升 → 回落 + SPX 同时下跌 → 反弹**。两个 addon 都触发时 Greek 符号反向（vega/gamma/delta 全反），见 [addon_greek_orthogonality](../../doc/addon_greek_orthogonality_2026-05-12.md) §3。

> **2nd Quant Review Q7.4 caveat（必读）**：**Greek sign opposition supports non-redundancy, but does not guarantee PnL hedge effectiveness**。vega exposure strike location 不同、gamma profile 不同、notional/BP/premium size 不同——不能仅凭符号反向就推断 portfolio 风险缓和。`Greek directions are opposite and therefore the two sleeves are not mechanically redundant. However, Greek sign offset is not a reliable hedge claim because magnitude and strike localization differ.`

## Verdict

| 问题 | 答案 |
|---|---|
| 两个 addon 重复吗？ | **否**。日级重叠 0.9%，事件级 74-86% 异步 |
| 该合并吗？ | **不应合并**。Greek 正交 + signal 正交 + 历史触发集异步 |
| 该竞争淘汰一个吗？ | **不应竞争**。两者捕捉的是不同 regime 下的不同机会 |
| 该担心 same-event BP 冲突吗？ | **已由 [q042_gate.py](../../strategy/q042_gate.py) joint cap 处理**，combined Q042 BP ≤ 20%，不挤占 aftermath sleeve |
| Q042-B 与 Aftermath 80% co-fire 是问题吗？ | **不是 portfolio 问题**。B 与 aftermath Greek 仍反向；N=5 不构成强结论（per 2nd Quant Review Q7.2，treat as "monitor as sample grows"）|

> **2nd Quant Review 正式表述**：
> *Q066 provides sufficient evidence that SPEC-064 Aftermath and Q042 Drawdown Overlay are not redundant and should not be merged or forced to compete. Same-day overlap is negligible, Q042-A is mostly vol-quiet drawdown exposure not captured by aftermath, and most aftermath windows are vol-only events not captured by Q042. However, the conclusion should be framed as "low-overlap / non-redundant," not "fully orthogonal." Greek sign opposition supports non-redundancy but does not prove portfolio hedge effectiveness. Co-fire downside scenarios remain possible and should be monitored, but they are not blockers to keeping both addons.*

## Caveats

- Q042 state machine 用 mock 30 / 90 TD hold，与生产 [signals/q042_trigger.py](../../signals/q042_trigger.py) 的 `active_position_expiry` 跟踪略简化。但 **trigger 计数**应近似正确（A=35, B=5 与历史预期相符）
- ±5 TD 窗口是 primary metric；±10 TD 作为 secondary cluster metric monitor only（per 2nd Quant Review Q7.1）。扩到 ±10 TD 不会推翻"不重复"结论，仅表征 broader stress/recovery cluster 关联性
- Q042-A 在 2013-2015 高频触发（每年 1-3 次）期间无 aftermath 同伴；如果未来 VIX 长期低位 + SPX 渐次回撤 是新常态，Q042 单独承担 alpha 来源是合理的
- 仅看触发频率，未量化触发日的 **回报相关性**。如果两个 addon 在 co-fire 日 PnL 高度相关，仍可能存在 portfolio 层风险叠加。**2nd Quant Review Q7.5 verdict: NOT FATAL**——本次 PM 问题是"是否功能重复"，触发频率 + signal space + payoff structure + Greek direction 已足以下"不合并"结论；但若 Q042 sleeve 进入实际 paper trading / scale up cap，应启动 Q067 PnL correlation 研究

### Co-fire downside scenario（per 2nd Quant Review Q7.3，必读）

存在同向亏损路径，**不是 partial hedge**：

```text
T0:  VIX peak → off-peak → Aftermath IC enters
T1:  SPX drawdown + MA10 reclaim → Q042 enters
T2:  Second-leg selloff:
       VIX re-expands
       SPX breaks lower
       → Aftermath IC put-side stress (loss)
       → Q042 call spread decays / expires worthless (loss)
       → BOTH sleeves lose simultaneously
```

机制：
- Q042 long vega 可帮一部分 IV expansion，但若 SPX 跌幅大 call spread delta/intrinsic 损失主导
- Aftermath IC short vega + short gamma 在 adverse move 下非线性扩大
- 两个 addon 在此路径下 PnL 同向亏损

历史 candidate：2008-09 雷曼时期、2020-02 → 2020-03 COVID 加速段、2022-04 → 2022-05 双重下跌段。N 小但 conceptually real。

**Greek sign opposition does not eliminate this path risk**。这不要求合并或关闭任一 addon，但作为正式记录的 portfolio failure mode（未来 monitoring 项），若 live event 中观察到 co-fire 同向亏损，应立即触发 Q067 PnL correlation 研究。
