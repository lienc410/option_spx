## TL;DR

**4 个组合的 marginal value**：

| Δ vs A (Main only) | Full 19y | Post-2020 | Recent 2y |
|---|---|---|---|
| **B (+ DD Overlay)** | +$118k / +1.3pp Ann ROE | +$83k / +4.5pp | +$54k / +9.3pp |

| **D (+ Both)** | +$171k / +1.4pp | +$118k / +4.5pp | +$57k / +9.8pp |

**关键结论**：
1. **DD Overlay 是主要 alpha 来源**——post-2020 单独 +4.5pp Ann ROE，recent2y +9.3pp
2. **Aftermath 的 marginal value 小且在收窄**——recent2y 仅 +$3.5k / +0.7pp
3. **Combinations 几乎完全线性可加**（D−A ≈ (B−A) + (C−A)），说明 daily-P&L 层面没有显著交互——但**这是 linear-distribution 近似的局限**，真实 stress tail 交互必须看 P4C blocked-entry / P4C.7 合成测试
4. **Tail trade-off**：B 与 D 都把 worst 20d 恶化 $3.4-4.4k（vs A），说明 DD Overlay 提供 alpha 的同时确实加深了局部尾部

## 2. 解读

### 2.1 DD Overlay 是后 2020 alpha 引擎

- Full sample +1.3pp Ann ROE
- Post-2020 +4.5pp Ann ROE
- Recent 2y +9.3pp Ann ROE

这与 Q062/SPEC-094 当时定 DD Overlay 部署的核心证据一致。Q072 P4A 在组合层验证了这个结论：**DD Overlay 不只是边际优化，是 alpha 主力**。

但 caveat：DD Overlay backtest 19y 是事后重放，2026-05-10 才实际部署。**实盘历史无此 alpha**。

### 2.2 Aftermath 的边际价值在 recent2y 几近消失

- Full +0.25pp Ann ROE
- Post-2020 +0.6pp
- Recent 2y 仅 +0.7pp（$3,500）

近年 Aftermath 触发数减少（recent2y 仅 1 笔 BPS_HV 入场，其它高 vol 期主策略用 IC HV 而非 BPS HV）。这呼应了 Q070 的结论："19y 仅 15 笔 BPS_HV，aftermath 标签并非稀缺约束"。

**但**：这不意味着可以关掉 Aftermath gate。Aftermath 的设计意图是**风险许可**（允许在 VIX peak-and-fall 后做 BPS HV），即使触发次数少也是低成本守护。P4A 不支持移除 aftermath gate。

### 2.3 Linear additivity 提示 daily-level 没有交互——但这是近似

D−A ≈ (B−A) + (C−A)：
- Full: $171 ≈ $118 + $53 ✓
- Post-2020: $118 ≈ $83 + $35 ✓
- Recent2y: $57 ≈ $54 + $3.5 ✓

**这是线性分布 P&L 近似的副作用**：trade exit_pnl 平均分配到持仓天数 → 不同 sleeve 的 P&L 在 daily 层不会"互相影响"。**真实 stress 期 spike-y losses 的同向叠加要靠 P4C blocked-entry counterfactual + P4C.7 合成 stress test 来量化**。

### 2.4 Tail 恶化是真实代价

B 和 D 的 worst 20d 比 A 恶化 $3.4-4.4k，max DD 恶化 $3.1k。peak BP 从 57% 升到 67.9%。

这是 DD Overlay 进入组合的成本：
- Worse worst 20d window: -$3,429 (full) / -$4,433 (post-2020 & recent2y)
- Max DD: -$3,121 (full)
- Peak BP: +10.9pp

→ 跟 Sharpe / Ann ROE 改善对比，trade-off 看起来**仍可接受**——但只要 backtest 没有真正的 second-leg case 验证。
