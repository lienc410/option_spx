## 1. Executive Verdict

> **Q072 concludes that global sleeve risk is real, but production governance should remain simple.**
>
> - **Do not** implement priority allocator
> - **Do not** implement static per-sleeve caps
> - **Implement** augmented default cap: normal default caps + stress episode tightening + second-leg short-vol block

**Q072 应回答的核心问题**："三个 sleeve（Drawdown Overlay / Aftermath/BPS_HV permission / HV Ladder）在 stress 期会 co-active 并争抢 BP；账户层应该用什么 cap / priority / 治理规则？"

**最终答案**：用 **Augmented Default Cap**——在 R1-R4 默认值基础上加 R5 stress-episode tightening 和 R6 second-leg short-vol block。

## 4. Production Recommendation — Augmented Default Cap

### Normal Regime (default)

```
R1: SPX PM pool cap           = 70% NLV
R2: /ES SPAN cap              = 80% NLV
R3: Combined economic cap     = 60% combined NLV
R4: Max short-vol exposure    = 50% combined NLV
```

19y backtest 几乎不 bind，不扰动 production。

### Stress Episode Throttle

```
R5: When tight stress episode active:
    SPX PM pool cap         = 60% NLV (down from 70%)
```

**R5 解决的是**：stress 期 sleeve stack 升高，但 second-leg 还未确认 —— 此时 reduce SPX cap 让 stress 后期的 entry 更精挑细选。

**Stress episode 定义**（沿用 P1 tight 口径）：
- VIX ≥ 22 OR
- SPX 从 20d/60d 高点回撤 ≥ 4% OR
- DD Overlay / Aftermath active
- 连续 stress-False ≥ 3 个交易日 → 退出

### Second-Leg Hard Stop

```
R6: When second-leg state active:
    Block all NEW short-vol entries
```

**R6 解决的是**：market 已从 recovery/chop 转为二次下跌，继续卖 vol / 卖 put 的风险收益恶化。

**Second-leg state 定义**（P4C.0 实现）：
- SPX 从 60d 高点回撤 ≥ 8%
- 过去 30 日内有 ≥ 2% 中段反弹（已 bounced）
- VIX ≥ 25
- Days since current episode start > 14
- 19y 历史仅 fire 26 天（0.5% of trading days），全部集中在 2008-09 和 2022

**Short-vol entries 范围**（必须 block）：
- HV Ladder new slot
- BPS_HV (Aftermath-permission-gated) new entry
- IC_HV / Bear Call HV new entry
- 任何 main strategy 的 short-vol 入场

**NOT 在 R6 block 范围**（允许通过）：
- Drawdown Overlay long-gamma 入场
- 任何 risk-reducing exit
- 任何 roll 操作（roll 是 risk neutral 或减仓）

### Eligibility-First, Priority-Never

Production 入场决策：
1. 通过 R1-R4 hard caps
2. 通过 R5 stress-aware tightening（若 stress episode active）
3. 通过 R6 second-leg block（若 second-leg state）
4. **No priority scoring layer**——FCFS 即可

## 7. Confidence & Caveats

**置信度**：高
- P1-P4 全套数据交叉印证
- 5-allocator simulation 给出 unambiguous "priority ≡ FCFS" 结论
- R6 second-leg block 在 2022 真实 backtest 中数值化保护证据

**主要 caveats**:
1. **Linear-distribution daily P&L 近似**：P3.3 co-loss lift 数值可能 understate（real MTM 会放大 worst-5% 但方向不变）
2. **DD Overlay / HV Ladder backtest 是事后重放**：2007-2025 期间 DD 实际仅 2026-05 后部署，HV Ladder 仍 pre-launch
3. **HV Ladder = V2f filtered placeholder**：Q071 lock 后 R2 cap 可能需调
4. **19y 无真正 4-sleeve second-leg 样本**：所有结论方向 robust，但 worst-case 量化依赖 backtest 重放
