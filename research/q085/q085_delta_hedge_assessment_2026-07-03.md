# Q085 附录 — Delta 对冲评估：去掉方向依赖后还有盈利空间吗？

**日期**: 2026-07-03（PM 提问当日回答）
**方法**: 对现有两套 BCD 合成交易记录做一阶归因分解——delta 分量 = 入场净 delta 0.40 × (出场 SPX − 入场 SPX) × 100；残差 = 总 PnL − delta 分量 ≈ **对冲版策略的税前收益**（theta/vega/gamma 合并项）。
**复现**: 见本文件同 commit 的计算（q082_p6_synth_trades.csv n=137；q083_p11 trades n=82）。

## 1. 分解结果

| | LOW_VOL 格 BCD (n=137) | Q083 carve 格 BCD (n=82) |
|---|---|---|
| 总 PnL | $1,016/笔 | $1,410/笔 |
| **delta 分量** | $532/笔（**52%**） | $750/笔（**53%**） |
| **残差（≈对冲后税前）** | **$484/笔** | **$660/笔** |
| 残差胜率 | **91%** | **90%** |

**核心事实**：账本恰好是"一半方向、一半 carry"。carry 半边异常稳定（90-91% 胜率）——这就是对冲后剩下的引擎。

## 2. 对冲摩擦（决定性的部分）

- **对冲规模**：1 张 SPX BCD 净 delta 0.40 → 美元 delta = 0.40×100×SPX ≈ 8 张 MES（与指数水平无关的常数）。
- **交易成本**：初始 8 张 + 到期平 8 张 + 持有期 gamma 再平衡 ≈ 10-12 个 MES 回合/笔 ≈ **$60-100/笔**——可承受。
- **保证金（致命项）**：8 MES × ~$2,300 ≈ **+$18k/笔占用**。BCD 本身 debit 今日尺度 ~$22k → 对冲版**每笔占用现金近乎翻倍（~$40k）**。
- **运营**：逐日盯 delta 再平衡，与 PM 每天 1 小时的时间预算直接冲突（当前全系统是收盘信号 + 手动执行）。

## 3. Cash-bound 判定（今日尺度）

| | 不对冲 | 对冲 |
|---|---|---|
| 每笔期望（今日尺度线性缩放） | ~$2,000+ | ~$900-1,300（残差减摩擦） |
| 每笔现金占用 | ~$22k | ~$40k |
| **每占用美元回报** | 基准 | **≈ 降为 1/3 至 1/4** |

对 cash-bound 账户，每占用美元回报是硬约束指标（per Q081 框架）。**结论：有盈利空间（carry 半边真实且稳定），但按占用现金回报计算急剧劣化，且运营模式不兼容——不建议作为主线。**

## 4. 方向依赖的更便宜削减路径（替代答案）

1. **结构路由已在做**：矩阵在 NEUTRAL 态路由 IC（29% 的可交易日本来就是 delta 中性结构）。
2. **Q085 本体就是答案**：S5 槽位（信号驱动 0.5x/1x 规模）= 用信号质量条件化 delta 敞口——方向信号差时少拿方向，而不是永远不拿方向。
3. 极端 regime 的**静态**尾部对冲（买跌期权/领口）可作为未来独立问题，与逐日动态对冲是不同的成本结构。

## 5. Caveats（留档）

- 一阶线性归因忽略 gamma 路径：真实对冲 PnL 随再平衡频率变化，52/48 的分割是估计不是精确值（方向性稳健）。
- 残差的 9-10% 亏损笔来自 vol crush + gamma，对冲不消除这些。
- 若未来现金规模显著扩大（cash-bound 约束松弛）或引入自动化执行，本判定的两个决定性前提消失，可重开评估。
