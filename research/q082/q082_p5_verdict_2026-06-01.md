## What this revision changes

Original P5 used forward SPX return distribution as a proxy to estimate
**BCD-vs-QQQ** comparison projected to 26y. Reviewer correctly flagged
that this proxy fails in two directions:

1. **SPX forward return ≠ BCD PnL**: BCD is path-dependent (diagonal,
   short-leg punch-through risk, IV path matters). The map "UP/FLAT/DOWN
   forward window → BCD PnL" is NOT monotone. Two windows both labeled
   "UP" can have very different BCD outcomes depending on path.
2. **SPX forward return ≈ QQQ return**: QQQ ≈ SPX/NDX beta. Using SPX
   forward as "proxy for BCD" + "actual for QQQ" collapses both sides of
   the comparison into the same quantity → BCD-vs-QQQ diff is
   structurally near-zero by construction, V1/V3 lose discriminating
   power.

Per reviewer: original V1 ("B-1 stronger") is therefore **artifact** —
exactly the kind of false negative ("declare BCD has no problem") that
G2 of Q081 already flagged. Reviewer recommends Option A: descope P2 to
the narrow fact-claim that the proxy CAN honestly support.

## Revised P5 verdict — Option A (sample representativeness only)

### Fact-claim (what P2 data DOES support)

Across 1,747 BCD-eligible days in 26y of signal history (matrix-routed
"bull_call_diagonal" by current selector — denominator confirmed clean
in §below), the forward 34d SPX return distribution is:

| Sample | n | %UP (>+1%) | %FLAT (±1%) | %DOWN (<-1%) | mean fwd SPX | p05 fwd SPX |
|---|---:|---:|---:|---:|---:|---:|
| 26y baseline | 1,747 | 52.0% | 23.4% | 24.6% | +0.63% | -4.11% |
| Q081 3y sample | 21 | 47.6% | 9.5% | **42.9%** | +0.52% | -3.56% |
| Δ (26y − 3y) | | +4.4pp | +13.9pp | **−18.3pp** | +0.11pp | −0.55pp |

**Fact**: Q081's 3y sample experienced 18.3pp more DOWN forward windows
than the 26y BCD-eligible baseline. The 3y was an **atypically
down-tilted** sample.

### What this DOES NOT support (per reviewer)

- "B-1 is stronger than Q081 P5 implied" — REJECTED. The projection from
  3y per-stratum BCD-vs-QQQ to 26y direction mix is unsound (proxy
  collapse + path-dependence).
- "BCD 26y aggregate edge would be ~+9.9pp vs QQQ" — REJECTED. Cannot be
  computed from forward SPX direction alone.
- Any quantitative ratification or refutation of B-1's strength.

### What this DOES support

- The 3y sample observation that "9/21 = 43% of BCD entries hit DOWN
  windows" is **not the typical regime profile**. Historically, that
  share is 24-25%.
- Therefore Q081's down-window underperformance evidence (-3.4pp vs QQQ
  in DOWN stratum, only 2/9 wins) is computed on a sample that
  over-represents the DOWN bucket.
- **Direction of B-1's residual risk**: in a typical 5-year forward
  window, BCD would encounter fewer down-window events than 2023-2026
  did. But how that translates to BCD-vs-QQQ aggregate edge requires
  actual BCD PnL data, not forward-SPX direction.

## Denominator audit (resolves reviewer's ADD concern)

Reviewer flagged 32 "IVP_HIGH" days (where IVP percentile ≥ 67) as
potentially mis-routed (per Q081 §D's qualitative argument that IVP_HIGH
should route to BPS). Investigated:

**Resolution**: The 32 days are correctly routed. The matrix gates by
`iv_signal` ∈ {HIGH, NEUTRAL, LOW} which is **NOT identical to** IVP
percentile bucket (IVP_HIGH = IVP ≥ 67). Sample inspection: days with
IVP=68 in 2006-2007 have `iv_signal=NEUTRAL` per the signal cache,
correctly routing to "LOW_VOL × NEUTRAL × BULLISH" → BCD cell.

**Matrix gate is not leaking.** The 1,747 denominator is post-full-
matrix-gate (regime + iv_signal + trend all applied). No exclusion
needed.

Note: Q081 §D's qualitative argument was about IVP_HIGH ROUTING to BPS;
it was a hypothetical recommendation, NOT a deployed matrix change. The
matrix today routes ALL iv_signal levels in LOW_VOL × BULLISH to BCD.
This is unchanged status quo.
