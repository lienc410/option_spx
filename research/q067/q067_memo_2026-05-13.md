# Q067 — IVP > 55 Gate: Threshold Jitter + Window Length Sensitivity

**Date**: 2026-05-13
**Owner**: Quant Researcher
**Status**: 完成；**结论：jitter 与窗口敏感性都比预期严重；建议下一步测试"收紧式 hysteresis"（不放宽 block，而是更严格地 unblock）**

---

## Question

2nd Quant 提出 IVP 55 阈值附近可能存在 rank-jump 噪声：VIX 17.6 → IVP 54、VIX 18.1 → IVP 65 可发生在 0.5 VIX 点之内。PM 额外提出窗口长度敏感性（1yr 与 6mo / 2yr 给同一 VIX 序列不同的 IVP）。

Q063 Phase 5 已确认在 19 年样本下：simple `IVP > 55` gate 优于 multi-factor / 放宽变体。但有两个 operational 问题未量化：

1. **Jitter 实际有多严重**：候选 entry 在 [50, 65] 附近的频率、阈值跨越的 flip-flop rate
2. **窗口长度选择本身的影响**：126 / 252 / 504 不同窗口给出多少 % 的决策分歧

## TL;DR

| 维度 | 结果 |
|---|---|
| Rank-jump artifact | **实证确认**：VIX median in IVP [55, 60) 是 17.27，**比 [50, 55) 的 17.85 还低** —— 穿过 55 时 actual VIX 可以下降 |
| Daily flip rate (19yr) | **7.37%** |
| Daily flip rate (最近 1 年) | **11.5%** —— 恶化 |
| 61% 的 flips 是 5 TD 内反转 (flip-flop) | 219 / 359 |
| Window 126 vs 252 决策分歧 | **15.15%** TD |
| Window 252 vs 504 决策分歧 | **15.60%** TD |
| Window 分歧落在 jitter zone [40,70] 比例 | **69.8%** |

**最关键发现**：穿过 55 阈值时 actual VIX 可以下降（17.85 → 17.27）。这是 rank-jump 实证存在的证据，不是理论假设。

**操作含义**：当前 hard gate 在 live 中会有约 7% 日子（recent: 11.5%）出现决策切换，其中 61% 在 5 TD 内反转。但 candidate entry 日子远少于交易日总数，所以实际触发频率比这低。**Jitter 严重程度足以触发 Phase 2 研究测试 hysteresis 变体**。

---

## Method

[q067_ivp_jitter_window_sensitivity.py](q067_ivp_jitter_window_sensitivity.py) 计算 4871 trading days (2007-01-03 → 2026-05-13) 的 3 个 IVP 窗口（126d, 252d=prod, 504d）：

- **A. Jitter zone density**：IVP 落在 [50, 65] 的 TD 占比，以及生产 IVP_252 各 band 对应 VIX 分布
- **B. Flip rate**：daily block ↔ allow 切换 + ±2/±5 TD cluster 频率 + 5-TD 内反转 (flip-flop) 比例
- **C. Window pairwise**：126/252/504 两两比较 block 决策分歧 TD 数
- **D. Joint**：窗口分歧是否集中在 jitter zone
- **E. Recent year**：2025-05 → 2026-05 的近期 jitter 强度

数据：yfinance ^VIX 2003-01 → 2026-05-13（5878 TD raw，warmup 后 4871 TD 进入分析）。

## Findings

### A. Jitter zone density

|  | in [50, 65] | above 55 | total |
|---|---|---|---|
| ivp_126 | 592 (**12.2%**) | 1949 (40.0%) | 4871 |
| ivp_252 (prod) | 625 (**12.8%**) | 2103 (43.2%) | 4871 |
| ivp_504 | 644 (**13.2%**) | 2249 (46.2%) | 4871 |

约 **1/8 of all TD** IVP 在 jitter band。Recent year 提升至 **19.2%**（VIX 长期压在 15-18 的副作用）。

#### 生产 IVP_252 各 band 的 VIX 分布（关键表）

| IVP_252 band | n TD | VIX min | VIX median | VIX mean | VIX max |
|---|---|---|---|---|---|
| [0, 50) | 2572 | 9.14 | 15.24 | 16.29 | 39.18 |
| [50, 55) | 196 | 11.04 | **17.85** | 19.15 | 42.04 |
| [55, 60) | 240 | 11.03 | **17.27** ← lower! | 18.81 | 43.23 |
| [60, 65) | 189 | 11.10 | 17.50 | 19.08 | 44.14 |
| [65, 70) | 234 | 11.68 | 16.70 | 19.19 | 45.54 |
| [70, 100) | 1392 | 11.47 | **23.53** ← real elevation | 26.03 | 76.45 |

**Rank-jump artifact empirically 确认**：[55, 60) 的 VIX median (17.27) 比 [50, 55) 的 (17.85) **更低**。Block decision 在 55 翻面时，actual VIX 可以是下降的。直到 IVP > 70 才出现 VIX 真正离开 17-18 密集区（median 跳到 23.53）。

### B. Threshold flip rate

- **Total daily decision changes**: 359 / 4871 TD = **7.37%**
- **Flips with another flip within ±2 TD**: 252 / 359 = **70.2%**
- **Flips with another flip within ±5 TD**: 312 / 359 = **86.9%**
- **Tight flip-flop (reverse within 5 TD)**: 219 / 359 = **61.0%**

7.37% 听起来低，但 219 个 flip-flops（block→allow→block 在 5 TD 内）证明：阈值附近极不稳定。**Recent year (260 TD) flip rate: 11.5%** — 显著恶化。

### C. Window pairwise disagreement

| 对 | disagree TD | % |
|---|---|---|
| IVP_126 vs IVP_252 | 738 | **15.15%** |
| IVP_126 vs IVP_504 | 1290 | **26.48%** |
| IVP_252 vs IVP_504 | 760 | **15.60%** |

#### 方向分解（生产 252 vs alt）

| Case | IVP_252 = prod | IVP_126 = 6mo | Count | % |
|---|---|---|---|---|
| both block (agree) | block | block | 1657 | 34.0% |
| prod block only | block | allow | **446** | **9.2%** |
| 6mo block only | allow | block | **292** | 6.0% |
| both allow (agree) | allow | allow | 2476 | 50.8% |

6mo 窗口比生产更**松** 6.0%、生产比 6mo 更紧 9.2%（生产含 2020 COVID 等更老高 VIX 数据，current value rank 更低，反而 less block 在某些日子；但 6mo 仅看最近 → tighter clustering，current vs cluster 提升 block 概率）。

| Case | IVP_252 = prod | IVP_504 = 2yr | Count | % |
|---|---|---|---|---|
| both block | block | block | 1796 | 36.9% |
| prod block only | block | allow | **453** | 9.3% |
| 2yr block only | allow | block | 307 | 6.3% |
| both allow | allow | allow | 2315 | 47.5% |

2yr 比生产更**松**——更宽窗口稀释 current value rank。

**业务含义**：选 6mo 比生产多 block ~3pp 的天数；选 2yr 少 block ~3pp。**窗口选择本身就能产生 3pp 的 entry frequency 差异**——这与"55 阈值是否经济意义"无关，是 percentile 度量内在选择问题。

### D. Window 分歧位置

6mo vs prod 738 个分歧 TD 中，**69.8% 落在 IVP_252 [40, 70]**。
- **窗口分歧与 jitter 是同一现象的两个切面**：当 IVP 远离 jitter zone（< 40 或 > 70）时两个窗口大概率同意；分歧主要发生在 percentile 密集变化区
- 解决一个等于减轻另一个

### E. Recent year (2025-05-01 → 2026-05-13, 260 TD)

- IVP_252 in [50, 65]: 50 / 260 = **19.2%**（vs 19yr 12.8%）
- Daily decision changes: 30 / 260 = **11.5%**（vs 19yr 7.37%）

**最近一年 jitter 强度高于历史基线**——因为 VIX 大部分时间压在 14-18 极窄区间。若 VIX 持续 compressed，jitter 问题会**继续恶化**。

## Verdict (refined from 2nd Quant analysis)

### 同意 2nd quant 的核心论点：
1. ✅ **IVP 55 不是经济悬崖**（rank-jump empirically 证实）
2. ✅ **保持 simple hard gate 直到有更多证据**（Q063 Phase 5 实证支撑）
3. ✅ **不应放宽阈值**（放宽即接受过去 Q063 Phase 5 已否决的更松变体）

### 但量化结果支持**升级 standing monitoring 到 standing research task**：
- Jitter rate 7.37% historical / 11.5% recent 已超过 "monitor" 阈值
- 219 个历史 flip-flop 案例足以测试 hysteresis 变体
- 窗口选择产生 ~15% TD 决策分歧，**生产 252d 应在 SPEC review 中显式记录为 "default choice, not optimal"**

### Phase 2 推荐研究（条件触发）

**触发条件**：live 中 candidate entry 落在 IVP [50, 65] 区间出现首次 outcome（无论 win or loss）

**Phase 2 scope**：
1. 量化 candidate entry 落在 [50, 65] 的实际频率（不是 daily 而是 entry 频率）
2. 测试 **"收紧式 hysteresis"**：block if IVP > 55，unblock only if IVP < 50 持续 N TD（N=3, 5, 10）。**不是放宽 block，是更严格 unblock**
3. 测试 ivp63 / ivp252 联合 gate：BPS_NNB block if ivp252 > 55 AND ivp63 > 50（要求 short-window 也确认）
4. 跨窗口稳定性：block if **任一**窗口 > 55（最保守）vs **多数**窗口 > 55（中间）

每个变体在 19yr 样本下 vs 现 hard gate 量化：trade count、PnL、win rate、worst trade。
