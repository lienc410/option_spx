# Q067 Phase 2 — Hysteresis / Multi-Horizon / Cross-Window Gate Variants

**Date**: 2026-05-13
**Owner**: Quant Researcher
**Status**: 完成；**结论：5 个变体全部 FAIL strict-dominance；hard gate IVP_252 ≥ 55 是实证 local optimum**

---

## Question

Q067 Phase 1 量化 IVP > 55 gate 的 jitter（daily flip 7.37%、61% 5-TD 反转、最近 1 年恶化至 11.5%）。Phase 2 测试三个变体方向能否在不放宽 block 的前提下减少 jitter：

1. **V1 收紧式 hysteresis**：block if IVP ≥ 55；unblock only if IVP < 50 for N TD（N=3/5/10）
2. **V2 multi-horizon agreement**：block if IVP_252 ≥ 55 AND IVP_63 ≥ 50
3. **V3 cross-window stability**：block if ANY of (IVP_126, IVP_252, IVP_504) ≥ 55

## 成功标准（PM 预设）

- Historical Ann ROE: ≥ baseline - $750/yr (~0.5pp on $150k 账户)
- Daily flip rate: < 3%（vs baseline 7.89%）
- Worst trade: 不恶化

## Findings

### 1. V1 Hysteresis — 收紧 unblock 单调换 alpha for stability

| 子变体 | block days | flip rate | flip-flop | bps_n | Δ Ann | 决策 |
|---|---|---|---|---|---|---|
| V0 (hard 55) | 3687 / 8653 (42.6%) | 7.89% | 416 | 38 | — | ref |
| V1a (N=3) | 4460 (51.5%) | 4.15% | 122 | 15 | -$2,433 | ❌ |
| V1b (N=5) | 4859 (56.2%) | 3.41% | 60 | 5 | -$3,920 | ❌ |
| V1c (N=10) | 5556 (64.2%) | 2.32% | 20 | **0** | -$4,194 | ❌ |

观察：
- **Flip rate 监督下降 7.89% → 2.32%**（N=10）—— 机械上有效
- **但 alpha 单调下降**：N 越大 block 总天数越多，BPS NNB 进入次数越少，alpha 流失加剧
- **V1c (N=10) bps_n=0**：彻底阻挡所有 BPS NNB trade → 伪 "worst trade $0"（sample size=0）

**根因**：V1 设计内在矛盾。要求"更严格 unblock"等于"一旦 block 就更难解除"等于"block 总量增加"。这不是纯 stabilization，而是 **stability + 额外保守化 同时发生**。$2.4k - $4.2k/yr 的 alpha 损失是 stability 提升的隐性 cost。

### 2. V2 Multi-Horizon — 是 "放宽 block" 的伪装

| 指标 | V0 baseline | V2 multi-horizon |
|---|---|---|
| Block 条件 | IVP_252 ≥ 55 | IVP_252 ≥ 55 **AND** IVP_63 ≥ 50 |
| Block days | 3687 (42.6%) | 2790 (32.2%) ← 少 block 10pp |
| BPS_n | 38 | 48 (+10) |
| Ann PnL | $26,104 | $27,060 (+$956) |
| Worst trade | -$9,379 | -$12,021 (恶化 $2,642) |
| Flip rate | 7.89% | **8.74%** (恶化) |

**V2 的本质是放宽**：AND 联立两个条件 → 更难触发 block → block 天数下降 10pp → 更多 entry。

- PnL +$956 来自更多 trades，但 worst trade 同时恶化 → 是 alpha + risk 同步放大，不是 free lunch
- **Flip rate 反而恶化**：两个 horizon 各自抖动叠加触发 → 联合判断不稳定
- 这变体 **实质落入 Q063 Phase 5 被否决的"relax block"领域**

### 3. V3 Cross-Window Any — 三窗口叠加导致 flip 加剧

| 指标 | V0 baseline | V3 crosswin any |
|---|---|---|
| Block 条件 | IVP_252 ≥ 55 | ANY of (126/252/504) ≥ 55 |
| Block days | 3687 (42.6%) | 4777 (55.2%) ← 多 block 13pp |
| BPS_n | 38 | 10 (-28) |
| Ann PnL | $26,104 | $22,631 (-$3,473) |
| Worst trade | -$9,379 | -$9,379 (unchanged) |
| Flip rate | 7.89% | **8.76%** (恶化) |

**反直觉但成立**：三窗口"任一触发即 block"应该最稳，但 flip rate **最差 8.76%**。原因是三个独立 percentile series 各自抖动，OR 联立放大了任何单一抖动的影响。

实证结论：**多 source 联立 ≠ 更稳定**，可能反而 noisier。这与 V2 的发现一致：联立 gate 在 percentile 跳变密集区是 amplifier，不是 filter。

## Verdict

### Phase 2 综合结论

**5 个变体没有一个 strictly dominate baseline**。Q067 Phase 2 与 Q063 Phase 5 一致：**simple hard `IVP_252 ≥ 55` gate 是 19yr 样本下的 empirical local optimum**。

### 三个方向各自的死因

| 方向 | 死因 |
|---|---|
| V1 hysteresis | Unblock 收紧 → block 总量增加 → alpha 流失（线性单调）；reduce jitter 但不是 free lunch |
| V2 multi-horizon | 联立条件实质放宽 block → 进入 Q063 Phase 5 否决领域；flip rate 也恶化（两 horizon 抖动叠加）|
| V3 cross-window any | 三 percentile series 独立抖动 → OR 联立放大 flip → flip rate 反而最差 |

### Phase 2 启示

1. **Jitter 是 percentile 度量内在性质**，threshold-based gate 无法在维持 alpha 同时消除
2. 任何 "stability + alpha + risk" 三 KPI 同时改善 的变体在当前框架下都不存在
3. 若要彻底解决 jitter，需要 **非 threshold-based** signal（如 smoothed IVP、derivative-based regime detection），那是 Q063/Q067 框架外的新研究方向

### PM 决策选项（按推荐度排序）

| 选项 | 含义 | Quant 推荐度 |
|---|---|---|
| **A. 接受 Phase 2 verdict，关闭 backlog，保持 V0 hard gate** | 不动 production；Q067 + Q063 Phase 2 共同 close | ✅ 强烈推荐 |
| B. PM 愿意付 $2.4k/yr alpha 换 flip 减半 → V1a (N=3) | 操作 stability 优先于 micro alpha；BPS_n 38→15 大幅减少 | 🟡 看 PM 风险偏好 |
| C. 进一步探索非 threshold-based signal | 新研究 Q068，例如 smoothed IVP 或 regime detection | 🟡 高工程量，不一定有 alpha |
| D. 接受 V2 +$956/yr PnL，忽视 flip + worst trade 恶化 | **不推荐** —— 实质落入 Q063 Phase 5 否决领域 | ❌ 避免 |

**我推荐 A**：Phase 2 双 phase 实证（Q067 + Q063 Phase 5）已经覆盖足够大的变体空间。没找到更优解就接受 local optimum，比强行迁就 jitter 更稳健。

## Caveats

- Strict-dominance 成功标准（PnL ≥ -$750/yr、flip < 3%、worst 不恶化）是我提议的；若 PM 改用 weighted utility（如愿意付 $X 换 flip 减少 Ypp），V1a 可能改判
- $150k 账户尺寸下 $2.4k/yr 约 1.6% Ann ROE；其他账户尺寸结论可能微调
- 仅测试 5 个 representative 变体；其他 N 值 / hybrid 组合未测
- BPS NNB path 之外的 selector path 未受影响；Phase 2 仅评估这一 entry pathway
- 测试样本 19yr (2007-2026)；未做 OOS split（Q063 Phase 5 已确认 OOS 一致性，Phase 2 直接 follow）
