## 0. 问题定义（最高准则）

这是**带方向偏好的清仓执行（liquidation execution）问题**，不是信号问题：

- 一个 ¥908k、10 只 A股公募基金组合，**确定逐步清仓换美元**（非可选，时间模糊）。
- 约束：① 不砸在局部底；② 不机械"为卖而卖"；③ 不因"等不到好点"而无限死扛。
- **最高准则（2026-05-31 重订）**：一切按**经济意义最优**判断（出场质量、$/风险、tail）。无 backtest、阈值是先验，但**禁止**用"简单/省工程量/可解释"偏好现有或更简方案；可解释性是采纳因素**之一**（PM 不信任=零价值），需显式称重、不能用来否决一个经济更优项而不量化其代价。过度工程同样有经济代价 → 判**边际经济收益**，工程量不是约束。〔旧措辞"可解释性 > 理论最优"已废止〕

KPI 不是"卖在最高点"，而是 **实现均价 vs TWAP 基准的差 + 完成时间方差**。

## 2. 数据与口径

- 来源：akshare `fund_open_fund_info_em(symbol, indicator="累计净值走势")`。
- **用累计净值不用单位净值**：规避分红除权假跌。口径细节：累计净值"分红加回不复利"，对 ~1 年 MA 影响极小，对"持有以来最高"长区间有可见差异 → 输出标注口径。
- NAV = T+1、收盘后、无量价 → RSI(14) 仅弱参考，不赋择时权重。
- **数据长度红标**：024930、024915 为次新基，很可能 <60 交易日 → MA60/近60高/近250高 = NaN，**显式降级标红**，不静默出 NaN。
- akshare 无 SLA：retry(3) + sleep 节流 + 逐只 try/except 隔离，单只失败跳过并报告，不中断整体。

## 3. 每只基金指标

最新累计净值(+日期)、MA20、MA60、近60日最高、持仓以来最高(§4)、距MA20%、距MA60%、距近60高%、趋势三态、RSI(14)、**60日年化波动**(供 §5.4 分档)。

**趋势三态**：上升 ⟺ 最新>MA20>MA60；下降 ⟺ 最新<MA20<MA60；否则 震荡。
**趋势态用 band-hysteresis（现价在 MA20 ±1% 内视为"贴线"、不判方向）压日度 whipsaw**（§5.6）。

## 4. 追踪止盈参照高 = 滚动最高（不用 purchase_date）

规则④追踪止盈需要一个"前高"做锚（`现价 ≤ 现价×0.92` 永不成立 → 必须锚一个更高的高点）。

**决定（PM 2026-05-31）：锚用"滚动最高"，不引入 purchase_date。**
- 锚 = **近250日最高累计净值**（次新基用全可用历史最高）。完全由可观测净值算，PM 无需提供任何输入。
- 放弃"持仓以来最高 / purchase_date" 口径——违背"可解释、不加无谓输入"最高准则，且对实际持仓边际价值极低：
  - 华夏卓越 024930 次新基，全史≈持有期，无买入前旧峰可排除。
  - 睿远 007119 浮盈 +69%，买在远低于现价处，窗口内无"现价之上且买入之前"的峰。

**唯一残留风险**：仅当"买入 <250 天前且刚好买在更高峰正下方"时，滚动高偏高 → ④偏早触发。本组合 10 只均不符合（赢家买在低位、输家被②/③优先接管）；且④仅"减仓"非清光、有超卖否决兜底，伤害有限。

## 5. 出场规则 v2（按优先级，取首个命中）

参数：`STRONG=0.03`、`DEEP=0.15`、`TRAIL`=σ-scaled 连续 `clip(1σ·√30d,5%,14%)`（§5.4）。

| 优先级 | 条件 | 动作 | 卖出量(clip) |
|---|---|---|---|
| **① 上升趋势** | 趋势=上升（最新>MA20>MA60，**不论距高**·PM 2026-05-31 放宽） | **纯持有，让利润奔跑，零卖出** | **0** |
| **② 深套防砍底** | 趋势=下降 且 距近60高 ≤ −DEEP | 勿砍底，待反弹至阻力，**硬时限 6–8 周**，到期走保底量 | 保底/暂缓 |
| **③ 趋势转弱减仓**【新·填温水阴跌缺口】 | 趋势=下降 且 距近60高 > −DEEP（空头排列未深套） | 主动分批减（受超卖否决保护） | 保底+**明显加速** |
| **④ 追踪止盈** | 最新 ≤ 滚动最高(近250日/全可用史) ×(1−TRAIL_σ) | runner 出场（兜底地板） | **重仓出** |
| **⑤ 跌破MA20** | 趋势=震荡 且 MA20>MA60 且 最新<MA20 | 动能转弱，止盈减仓（受超卖否决） | 保底+小加速 |
| **⑥ 其余** | — | 持有观察 | 保底量(若属非强势) |

### 关键设计决定

1. **① 上升趋势 = 纯持有（PM 风险偏好定夺，2026-05-31 放宽至全部上升趋势）**。理由：基金按 NAV 赎回无滑点，"卖在强势抢流动性"的执行理由失效；PM 选择让动量跑，**主动接受"上升趋势一直持有"的惰性**。原"距近高≤3%才持有"的 cliff effect（最大仓 024930 因 0.6pp 越界被减仓）促使放宽——**只要 最新>MA20>MA60 就纯持有，不论距高**，`STRONG` 不再用于持有判定。卖出重心全部落到转弱信号 ③④⑤ → "让赢家跑、只在趋势破（跌出上升）时出"。代价：当前持有比例 ≈ 账户 60%（5 只 ¥544k）。
2. **④ 自相矛盾的原始判据已修复**：原 "趋势=上升 且 最新<MA20" 是死代码（上升⟺最新>MA20）。⑤ 改为正确的"震荡且均线仍多头但破短均"。
3. **③ 填补 ④→⑥ 的"温水阴跌"缺口**：确认空头排列但未深套的阴跌基金，原会滑进 ⑥ 被动持有 → 新③主动减，由超卖否决防砸底。
4. **整个"下降"态被 ②/③ 按 DEEP 干净二分**，互斥无重叠。

| 增强 | 裁定 | 落地 |

| **5.1 排程/保底时钟** | 保留(脊柱) | 每月至少出清 X%；**仅覆盖"非强势"仓**（强势①豁免——这是对 GPT review "不可被任何信号暂停"的局部 PM override）；不被 ③④⑤ 暂停 |
| **5.2 横截面排序** | 保留 | 连续打分(距近高/趋势强度/距MA) → 卖出队列，每周从顶部卖到保底量 |
| **5.3 超卖否决** | 保留+精修 | RSI<30 或 20日新低 → 否决 **③⑤** 主动卖；**④ 仅"近期赢家(近15日≥5日上升)"豁免**（其无其它出场地板），**非赢家(慢性下跌)触发④也受否决=不砸底**；保底时钟到期豁免(非强势)。〔2026-05-31 精修：原"④全豁免"会让 009010(−54%深套、单日 MA razor-thin 穿越假阳)在 20 日低被④砸 40%；改"runner-gated"后 009010 正确受否决〕 |
| **5.4 波动率归一** | **采纳(σ-scaled)** | `TRAIL = clip(1σ·√30d, 5%, 14%)` 连续随各基自身波动缩放。经济意义：回撤超过"正常 6 周波动"才出场。2026-05-31 按经济最优指令把两档升级为连续——固定档把 22% 波动的 013610 当低波→误杀 40%(¥41k)；σ-trail 修正之。vol 档仅留作 PM 参考标签，不再驱动 trail |
| **5.5 市场 regime** | **降级** | 宽基(沪深300/中证800)MA 状态仅作**输出栏上下文提示**，**不进决策、不暂停保底时钟**（防 regime 变成新死扛借口） |
| **5.6 趋势 whipsaw 过滤** | **采纳(band-hysteresis)** | 现价在 MA20 ±1% 内视为"贴线"、不判方向，压日度 churn（日度跑下经济价值更高）。选 band 而非 EMA(EMA 不解决价格穿越 MA 的 churn)、而非严格 N 日确认(会被单日反弹过度反应) |
| **5.7 赎回费阶梯** | **升级为硬约束** | 排程层硬约束输入：份额差几周入零费档 → 保底时钟主动让路。唯一确定性、无需 backtest 的真 edge |
| **TWAP 对照** | 升一等公民 | 每次决策同时输出「建议卖出量 / 纯 TWAP 等额量 / 偏离%」，PM 实时自检偏离最稳基准多远 |

## 6. 已知偏差与局限（主动 disclose）

1. **风险方向已翻转**：原设计风险是"卖赢家留输家"；PM 选择骑赢家后，保底时钟主要作用在**非强势仓（含恢复中的输家）**。保护机制：②的等待+硬时限、超卖否决，确保恢复中的仓不被砸在坏点。
2. **量级**：睿远 ¥141k + 华夏卓越 ¥152k ≈ **¥293k ≈ 账户 32%**，只要持续强势就长期不卖；极端情形清仓最多推进约 2/3 盘子。**PM 已知并接受。**
3. **无样本外验证**：阈值是先验，不可声称"优化"。脚本记录每次决策 + naive TWAP 对照，事后评估。
4. **次新基数据不足** → 部分基金只能走 ④/⑥。
5. **akshare 无 SLA**，需异常隔离。
