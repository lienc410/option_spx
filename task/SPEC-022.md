# SPEC-022: Sharpe Robustness Test — Temporal Stability & Confidence Intervals

## 目标

**What**：量化当前 Sharpe 的统计不确定性，建立更鲁棒的性能评估框架，落实 Warning C："不要过度解读 Sharpe，仅用作临时排名指标"。

**Why**（Prototype 实证，`SPEC-022_sharpe_robustness.py`，2026-03-30，26yr 386 笔）：

单一 Sharpe 数字会被引用为"策略质量"的精确度量，但实际上它对时间窗口极为敏感，且有较大的统计不确定性。

## 关键发现

### 发现 1：整体 Sharpe=1.54 的 95% CI 是 [1.18, 1.95]

26yr Sharpe 的 Bootstrap 95% 置信区间宽度为 0.76。这意味着即使是 26 年数据，Sharpe 的统计不确定性仍超过 ±0.38。在比较两个策略的 Sharpe 时，差异小于 0.5 应该被视为统计噪声。

### 发现 2：3yr Sharpe 的置信区间 [0.39, 1.96] 极宽，几乎无信息量

3yr (n=69) 的 Bootstrap CI 宽度达 1.56，意味着：
- 真实 3yr Sharpe 可能是 0.39（勉强正向）
- 也可能是 1.96（优秀）

**结论**：任何基于 3yr 以下窗口的 Sharpe 比较，在统计上是无意义的。

### 发现 3：年度 Sharpe 波动巨大（−0.92 到 +7.13），22% 的年份 Sharpe < 0.5

5 个负 Sharpe 年份（2004、2008、2011、2015、2022）分布在不同的市场背景下：
- 不是集中在某类特定市场（不只是熊市）
- 2004 是牛市但 IV 极低（BPS 机会少）
- 2008/2011/2015/2022 是高波动期

这说明系统在以下情形下 Sharpe 退化：a) IV 过低（机会少），b) 急速方向性行情（2022 Federal Reserve）。

### 发现 4：Bull Call Diagonal 是 Sharpe 最稳定的策略（26yr 1.69 vs 3yr 1.75）

这是整个策略族中唯一在 26yr 和 3yr 几乎相同的策略（+0.06 差异）。与 SPEC-016（最低 haircut）和 SPEC-018（最对称分布）一致。Diagonal 可能是整个系统中最"可靠"的 alpha 来源。

### 发现 5：BPS / IC_HV 的 3yr Sharpe 严重退化（分别 −1.93、−1.67）

BPS 从 26yr 2.50 降至 3yr 0.57（−1.93），IC_HV 从 2.22 降至 0.55（−1.67）。这与 SPEC-016（haircut 后 HV 信用策略真实 alpha 有限）和 SPEC-018（HIGH_VOL regime 3yr Sharpe 从 1.83 降至 0.60）完全一致。

三个不同角度（haircut、metrics、Sharpe robustness）得出同一结论：**HV 信用策略的 26yr 高 Sharpe 是历史特定时期（2000–2020 期间低波动牛市）的产物，不是稳健的结构性优势**。

## Sharpe 的正确使用指南

### 可以用 Sharpe 做的事

- **策略间相对排名**（用相同时间窗口比较）
- **系统整体健康检查**（Sharpe 突然从 > 1.5 降到 < 0.5）
- **粗粒度决策**（Sharpe > 1.0 vs < 0.5 是有效区分）

### 不应该用 Sharpe 做的事

- **基于 3yr 以下窗口的策略决策**（CI 太宽）
- **比较差异小于 0.5 的两个策略 Sharpe**（统计噪声范围内）
- **将 Sharpe 2.0 与 Sharpe 2.5 视为有意义差异**

### 推荐的补充指标（优先级高于单独 Sharpe）

| 指标 | 优势 |
|------|------|
| **正值年份比例** | 不依赖正态分布假设，直接可读（22/27 = 81%） |
| **Calmar ratio** | 把 drawdown 纳入计算，对 short-vol 系统更相关 |
| **最长连续亏损年数** | 对持有力 / margin 管理更有指导意义 |
| **负 Sharpe 的历史环境** | 知道在哪类市场下系统失效，比 Sharpe 本身更有价值 |

---

## 不在范围内

- 动态 Sharpe 实时计算
- 多因子风险模型
- 贝叶斯 Sharpe 估计
