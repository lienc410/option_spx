# SPEC-135 — Decision Trace：决策管道可视化（PM 需求 2026-07-07）

**需求**: 主矩阵 + 各类门 + 治理层日益复杂，PM 要"where are we / why we are here"的单屏可视化。当日实例：7/7 一天内路由三变（早 carve BCD+敞口降级 → 午仓位 ACTION → 收盘 G2 触发 reduce_wait）。

**反漂移铁律（本 SPEC 的存在前提）**: 禁参数镜像——图的结构与数值**全部由生产代码每次评估时自吐**（trace），前端零硬编码 gate 清单。手画流程图 = 被禁止的镜像文档。

## 0. 人类语言铁律（PM 2026-07-07 反馈，v2）

- **每个 trace 节点必须携带 `label_human`（主显示）与 `code_ref`（角标/悬停，如 SPEC-113）**——人类语言为一等公民，代号只做溯源。label_human **与门定义同居代码**（写在 gate 旁边，随 gate 改动同 commit），不建独立词汇表文档（反镜像）
- 文案标准：不预设读者知道任何 SPEC 编号/内部代号；数字必须带语义（"第 28 百分位（偏便宜）"而非 "IVP 27.9"；"已记录 2 天/需 60 天"而非 "n=2/60"）
- **期权/策略术语保留英文**（PM v3 确认）：Bull Call Diagonal、BPS、delta、DTE、call/put 等行业标准词不译；**解释性文字**用中文（如 "Bull Call Diagonal（买远月 call + 卖近月 call）" 首次出现时括注）
- **每个 ✓/⛔ 节点可展开（hover/点击，PM v3 需求）**：显示三件套 `{检查了什么数据, 实际值 vs 阈值, code_ref}`——如 "✓ 温和带条件" 展开为 "VIX 16.13 ∈ [15,18) ✓ · 趋势=BULLISH ✓（SPEC-113）"。trace 节点 schema 已含全部字段，UI 只做渲染
- Lane C 用叙事句式而非代号串（见 §3 示例）
- ④ 资金检查层按 **SPEC-131 v2 口径**渲染（策略资金池分母、阈值 30%、"不禁止仅改语气"语义）

## 1. Trace 结构化输出

`select_strategy()` 加 trace 累加器：每个评估节点 `{layer, check, input, threshold, outcome, branch_taken, reason}`——数据层（VIX/IVP/趋势读数）→ 体制分层 → 矩阵格路由 → 格内门（VIX RISING / IVP 上下限 / carve 条件 / SPEC-060 死格等，**逐个记录含静默通过的**）→ 治理门（SPEC-123 halts、D2 前置门）→ 敞口层（131）→ 最终输出与语气。`bcd_governance.evaluate_gates` 同样入 trace。**行为零变更**：trace 只追加不分支（AC：全历史回放路由 bit-identical）。

## 2. 存储与 API

- 每日 trace 追加 `logs/recommendation_log.jsonl` 既有行（扩展字段，strict-JSON）
- `/api/decision-trace?date=`（缺省当日；v1 支持最近 30 日切换）

## 3. 渲染（新 /pipeline 页或 /spx 折叠区，dev 定）

**三泳道垂直 rail**（走过的节点高亮、未走分支 ghost、每节点显示 实际值 vs 阈值）：

- **Lane A「今天开不开新仓？」** 五层：① 市场读数（VIX/期权贵贱分位/趋势，全部带语义描述）→ ② 查策略手册（矩阵格 + 特批通道，人话陈述条件）→ ③ 逐项安检（格内门链，**静默通过的也列**）→ ④ **资金检查（v2 新增层，PM "BP cap" 需求）**：现金池余额 vs 底线 → 本单/并发预算 cap（SPEC-111 链）→ 家族敞口占现金比（131）→ 允许张数 → ⑤ 安全刹车（治理门，含运行特性披露人话版）→ 今日结论
  7/7 实例渲染（人话版）：恐慌指数 16.1 正常区 → 手册允许"看涨对角价差"（波动 15-18 温和带特批）→ 安检通过 → ⛔ 刹停："该策略 18 个月合计 −$6,006 转负 → 暂停开新仓等复核（预注册说明：策略良好时每周期也有约四成概率误踩，是'要求复核'不是'宣布失效'）" → 今日观望
- **Lane B「手上的仓位要动吗？」**: 每个 open 仓位一行，人话陈述触发器状态（"卖出的近月腿只剩 10 天 → 规则要求平掉或滚动，已提醒"）
- **Lane C「地形参考——只描述，不决策」**: 叙事句式，例：「头顶期权持仓墙：7550 与 7600 价位堆着最多机构看涨合约，今天显著增厚（距收盘 +0.6%/+1.3%）；价格连续第 2 天贴墙——该状态正在积累记录，攒满 60 天做正式统计检验；历史价位簇与下降趋势线今天都不在价格附近」
